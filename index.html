Project Overview

What is OncoPedict?

An educational machine learning web app that predicts cancer risk based on clinical parameters. Built with a Python Flask backend and a pure HTML/JS frontend — no Node.js required.

How to Run

  • Open backend/ folder
  • Double-click run_backend.bat
  • Open index.html in browser
  • Select cancer type & enter values
  • Click Run Prediction
